# Message Encrypter 
- <h3>Programming Language:</h3> Python Programming Language
- <h3>Libraries: </h3> cryptography
- <h3>Cryptography Algorithms: </h3> AES-CBC, HMAC-SHA256

# How To Use:
- open terminal
- git clone https://github.com/Yarmm360/message-encrypter-cli.git
- cd message-encrypter-cli
- python message_encrpter.py
----
- enter message which you wanna encrypt.
- encryption key will be saved in txt file on your folder.
